
<h1 align="center">Hi, I'm Alexandre Fonseca (Arcani97).</h1>

I'm an electrotechnical technician and electrician in Rio de Janeiro, as well as a programming enthusiast and tabletop RPG lover. In my free time, I combine these passions by developing modules and translations for Foundry VTT. My goal is to create useful, system-agnostic tools that work for any game, filling a massive gap for universal modules in the community. Hope you enjoy my work, and if you'd like to support me, my Patreon link is right below!
